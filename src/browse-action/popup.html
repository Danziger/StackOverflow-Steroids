  • PRO EDITOR

    See your answer's preview side to side with what you are actually writing without scrolling the whole page.

    Note this will only work when page width is wider or equal to 1456px.

  • EDITOR'S OPTIONS

    • Block answer editor scroll

      Prevent page scrolling when the answer editor is focused (when using keys) or the mouse is over it (when using scroll wheel).

      You can still scroll it when the focus blurs or moving the mouse outside it.

    • Block code editor scroll

      Prevent page scrolling when the code editor is focused (when using keys) or the mouse is over it (when using scroll wheel).

      You can still scroll it when the focus blurs or moving the mouse to the margins around the editor popup.

  • EDITOR HELP

    • Stylish design

      Just some extra borders, delimited tabs and some other details to make it nicer.

    • Overlay editor's help

      The Markdown Editing Help bubbles won't displace the editor any more. They will just overlay over it.
